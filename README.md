## Installation :
- Back : `brew install maven`
- Front : `npm install`  

## Run :
- Back : `mvn spring-boot:run`
- Front : `npm start`


## Stack :
- **Back** : Springboot  
- **Front** : React
- **Design** : Figma
- **Database** : PostgreSQL

## About :
This project aims to offer an application in which parents can log in to register their children for extracurricular activities.
